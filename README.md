# Imob Parcerias

Plataforma B2B de parcerias entre corretores de imóveis — MVP (PWA).

## Stack
- **Frontend:** Next.js (React), Mobile-First, PWA → `apps/web`
- **Backend:** Node.js (API REST) → `apps/api`
- **Banco:** PostgreSQL (Neon serverless)

## Estrutura
```
imob-parcerias/
├── docs/
│   ├── desenvolvimento/   # documentação de desenvolvimento
│   └── referencias/       # arquivos auxiliares (PDFs de origem, logo)
├── apps/
│   ├── web/               # Frontend Next.js
│   └── api/               # Backend Node.js
└── packages/              # Código compartilhado (opcional)
```
